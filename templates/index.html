Student Math Score Analysis

Problem Statement

This project aims to understand how a student's math score is affected by other variables, such as Gender, Ethnicity, Parental Level of Education, Lunch, and Test Preparation Course.

Data Exploration

In this section, we will explore the given dataset to understand its structure and distribution of data in each variable. We will also handle any missing values appropriately.

Data Visualization

Here, we will create visualizations such as scatter plots, histograms, box plots, or bar charts to gain insights into the relationships between the math score and each independent variable.

Correlation Analysis

In this section, we will calculate the correlation coefficients between the math score and other numerical variables to identify which independent variables have a stronger or weaker linear relationship with the math score.

Categorical Variable Analysis

For categorical independent variables like Gender, Race/Ethnicity, Parental Level of Education, Lunch, and Test Preparation Course, we will use box plots or bar charts to compare the distribution of math scores among different categories.
